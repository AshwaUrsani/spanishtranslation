## 🎯 Project Overview

This project goes beyond training a model — it delivers a **fully working translation product**. A user types any English sentence into the web app and receives an instant Spanish translation, powered by a deep learning model I designed, trained, and deployed end-to-end.

The goal was to understand how large language models like Google Translate work under the hood, and then build a scaled-down version from scratch.

---

## 💡 What I Did

- **Designed the model architecture** — built a Transformer network from scratch using TensorFlow, the same foundational architecture behind modern AI translation tools
- **Trained on bilingual data** — fed the model thousands of English-Spanish sentence pairs to learn cross-language patterns
- **Solved a real deployment problem** — ensured the model's vocabulary is saved and reloaded correctly, so it behaves identically in production as it did in training
- **Shipped a live product** — deployed the model as an interactive web app on Streamlit Cloud, linked to GitHub for continuous access
